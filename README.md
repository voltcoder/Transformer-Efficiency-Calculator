#  Transformer Efficiency Calculator

A Python-based project to calculate transformer efficiency at different load conditions considering:

- Copper Loss (Variable)
- Iron Loss (Constant)
- Load Percentage

This project demonstrates practical implementation of transformer efficiency concepts used in Power Systems.

---

##  Formula Used

Efficiency (%) = Output Power / (Output Power + Copper Loss + Iron Loss) × 100

Where:

- Copper Loss ∝ (Load)^2
- Iron Loss = Constant
